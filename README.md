# Game Engine

A custom C++ game engine built entirely from scratch, using OpenGL for rendering and hand-written GLSL shaders. Designed for learning, experimentation, and full control over every part of the rendering pipeline.

## Overview

This project implements it's own
  - Collision System
  - Rendering System
  - Memory Management Solutions
  - Input System
    - Built on top of SDL's input
The engine performed at ~1500 fps when testing under a development environment.
